Need to add `console.log()`?
Type `log` and press <kbd>⇥</kbd> (macOS) / <kbd>Tab</kbd> (Windows/Linux).
Or first add the value, then type `.log` and press <kbd>⇥</kbd> (macOS) / <kbd>Tab</kbd> (Windows/Linux).
