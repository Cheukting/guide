In WebStorm, you can comment and uncomment blocks of code with <kbd>⌘⇧/</kbd> (macOS) / <kbd>Ctrl+Shift+W</kbd> (Windows/Linux). To do the same for lines, use <kbd>⌘/</kbd> (macOS) / <kbd>Ctrl+/</kbd> (Windows/Linux).
