Rename your classes, methods, and variables using **Refactor** | **Rename**. Press <kbd>⇧F6</kbd> (macOS) / <kbd>Shift+F6</kbd> (Windows/Linux), type the new name and press <kbd>⏎</kbd> (macOS) / <kbd>Enter</kbd> (Windows/Linux).
