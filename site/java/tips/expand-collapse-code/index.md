You can expand code blocks with <kbd>⌘+</kbd> (macOS) / <kbd>Ctrl++</kbd> (Windows/Linux) or collapse them with <kbd>⌘-</kbd> (macOS) / <kbd>Ctrl+-</kbd> (Windows/Linux).
