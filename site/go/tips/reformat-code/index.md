_Reformat Code_ makes it a piece of cake to fix style warnings and make your code formatted like _go fmt_ would.

**How to use:**

Invoke _Reformat Code_ via <kbd>⌘⌥L</kbd> (macOS) / <kbd>Ctrl+Alt+L</kbd> (Windows/Linux).
