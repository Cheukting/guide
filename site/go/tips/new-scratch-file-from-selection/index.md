**How to use:**

Select a portion of the code, press <kbd>⌘⌥⏎</kbd> (macOS) / <kbd>Ctrl+Alt+Enter</kbd> (Windows/Linux), then choose **Create new scratch file from selection** to quickly create a new scratch containing the same code.
