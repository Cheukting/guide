**How to use:**

Connect to a MongoDB database from the IDE, then open a _Console_ to it and write a query.

To run it, press <kbd>⌃⏎</kbd> (macOS) / <kbd>Ctrl+Enter</kbd> (Windows/Linux) to run the query and retrieve the results.
